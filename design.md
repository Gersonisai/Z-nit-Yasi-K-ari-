# Diseño Visual - YASI K'ARI Web Institucional

## Filosofía de Diseño

### Identidad Visual
- **Elegancia Profesional**: Diseño que refleja la seriedad y confianza de una firma legal establecida
- **Minimalismo Ejecutivo**: Limpio, sofisticado, con enfoque en la legibilidad y la experiencia del usuario
- **Identidad Boliviana**: Elementos sutiles que conecten con la cultura y geografía boliviana

### Paleta de Colores
- **Primario**: Azul marino profundo (#1a3a5f) - Confianza, profesionalismo
- **Secundario**: Azul intermedio (#2c5a8a) - Complemento y jerarquía
- **Acento**: Dorado suave (#e6a23c) - Elegancia y prestigio
- **Neutros**: Blancos, grises claros para fondos y texto

### Tipografía
- **Display/Títulos**: Playfair Display (serif elegante para encabezados)
- **Cuerpo/Texto**: Inter (sans-serif moderna y legible)
- **Alternativa**: Segoe UI como fallback según documentación

## Efectos Visuales y Animaciones

### Librerías Implementadas
1. **Anime.js**: Animaciones sutiles de entrada y transiciones
2. **ECharts.js**: Visualización de datos geográficos y estadísticas
3. **Splide.js**: Carruseles de imágenes y testimonios
4. **p5.js**: Elementos visuales interactivos sutiles
5. **Matter.js**: Efectos físicos en elementos decorativos
6. **Pixi.js**: Efectos visuales avanzados para backgrounds
7. **Shader-park**: Efectos de fondo sofisticados

### Efectos Específicos
- **Hero Background**: Shader de gradiente dinámico con partículas sutiles
- **Texto Animado**: Efecto de escritura para títulos principales
- **Cards de Servicios**: Hover effects con elevación y sombra
- **Mapa Interactivo**: Animaciones de marcadores y transiciones suaves
- **Scroll Animations**: Reveal effects sutiles al hacer scroll

## Layout y Estructura

### Sistema de Grid
- **Desktop**: 12-columnas con breakpoints responsivos
- **Mobile**: Single column con elementos apilados
- **Espaciado**: Sistema consistente de márgenes y padding

### Componentes Principales
1. **Header**: Navegación superior con logo y menú
2. **Hero Section**: Impacto visual inmediato con mensaje clave
3. **Servicios**: Grid de especialidades legales
4. **Ubicaciones**: Mapa interactivo de bufetes
5. **Equipo**: Perfiles de abogados con carruseles
6. **Contacto**: Formularios integrados con WhatsApp

## Responsive Design
- **Mobile First**: Diseño optimizado para dispositivos móviles
- **Breakpoints**: 320px, 768px, 1024px, 1440px
- **Touch Friendly**: Elementos interactivos de tamaño adecuado
- **Performance**: Optimización de imágenes y recursos

## Accesibilidad
- **Contraste**: Ratio mínimo 4.5:1 para texto
- **Navegación**: Teclado y screen readers
- **Alt Text**: Descripciones para todas las imágenes
- **ARIA Labels**: Etiquetas semánticas para elementos interactivos
